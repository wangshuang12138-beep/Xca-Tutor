# Xca-Tutor

> 你的私人英语陪练 Agent

## 项目概述

Xca-Tutor 是一个专注于英语口语、听力、词汇训练的 AI 陪练工具。通过语音交互，让用户在真实场景中练习英语，并提供详细的复盘报告和针对性重练。

## 核心理念

**"从用到学，从错到会"**

不是背单词，而是在真实对话中自然习得；不是避免犯错，而是从错误中快速进步。

## 技术栈

- **前端：** Swift + SwiftUI（原生 macOS 应用）
- **语音：** OpenAI Whisper API（识别）+ OpenAI TTS API（合成）
- **对话大脑：** OpenAI GPT-4o
- **数据：** SQLite（本地存储）
